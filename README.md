
# Sitemap de TiendaTech

Este repositorio contiene el diseño de la arquitectura de la información y el sitemap para la tienda online "TiendaTech".

## Estructura de Páginas
* **Página de inicio (/)**
  * **Productos (/productes)** -> Detalle de un producto (/productes/{id})
  * **Categorías (/categories)** -> Móviles, Computadoras y Accesorios
  * **Cesta (/cistella)**
  * **Cuenta de usuario (/compte)**
  * **Contacto (/contacte)** (Incluye formulario y mapa)

## Archivo del Diagrama
El archivo original en formato XML para abrir y editar en Draw.io se encuentra en la raíz de este repositorio con el nombre `tu_archivo.xml`.
